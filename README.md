# BoardGame Number Challenge

ボードゲームの補助アプリ。数字を漢数字・ローマ数字・アラビア数字でわかりにくく表示し、プレイヤーが制限時間内に読み取って物理ボードゲームに反映するチャレンジ型Webアプリ。

## 技術スタック

- Vite + React + TypeScript
- Tailwind CSS v4
- React Router v7
- Vitest

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## テスト

```bash
npm test
```

## ビルド

```bash
npm run build
```

## Lint

```bash
npm run lint
```
