# BoardGame Number Challenge

## ゲームの趣旨

ボードゲームの補助アプリ。物理ボードゲームのプレイ中に使用し、プレイヤーの「数字読み取り能力」を試すチャレンジを提供する。

### 遊び方

1. **トップページ**: 各モデル（Human / Opus / Sonnet / Haiku）ごとに表示する数字の個数を選択し、Start を押す
2. **ゲームページ**: 10秒間、数字が漢数字・ローマ数字・アラビア数字のいずれかの形式で、点滅・回転・移動のアニメーション付きで表示される。プレイヤーはこの数字を読み取り、物理ボードゲーム上で活用する
3. **結果ページ**: 表示されていた数字がモデル別・色別にアラビア数字で一覧表示される。答え合わせに使用

### モデル設定

| モデル | 色 | 数字範囲 | 個数 |
|--------|------|----------|------|
| Human | グレー | 8〜13 | 1固定 |
| Opus | 紫 | 6〜12 | 0〜5 |
| Sonnet | 青 | 3〜9 | 0〜5 |
| Haiku | 緑 | 1〜6 | 0〜5 |

### 難易度の仕組み

- 数字の表示形式（漢数字 / ローマ数字 / アラビア数字）がランダムに割り当てられる
- アニメーション（点滅・移動・回転・ドリフト）で読み取りにくくなる
- モデルごとに数字の範囲が異なり、多くのモデルを選択するほど画面上の数字が増えて難しくなる
